# Sales Data Analysis – Superstore Dashboard

## Project Overview
This project analyzes Superstore retail sales data to evaluate revenue trends, profitability, and regional performance using Tableau.

## Key Insights
• Technology category contributes highest profit (~51%)  
• West region generates highest revenue  
• Some states show negative profitability despite strong sales  
• Sales trend shows consistent growth over the years
